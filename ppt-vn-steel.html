越南最大私营企业集团，以资本与政商关系强势切入钢铁赛道

成立1993年，乌克兰起家
FY2024总资产839,216亿VND (~$33.6B)
FY2024营收191.96万亿VND (~$76.8亿)

业务板块地产/VinFast汽车/科技/酒店/教育
创始人Phạm Nhật Vượng（潘日旺）

VinMetal战略

2024年宣布投资VinMetal钢铁项目，规划产能500万吨/年，选址越南中部广义省Dung Quat经济区。

核心逻辑：
1. VinFast电动汽车 → 高端汽车板需求
2. 房地产/Vinhomes → 建筑用钢需求
3. 集团内需消化+外部市场扩张

预计投资规模$50-80亿，采用高炉-转炉长流程工艺。

风险与挑战

[R1] 钢铁行业零经验，技术积累从零起步

[R2] VinFast持续巨亏（2024年净亏~$24亿），集团现金流承压

[R3] 建设周期5-7年，和发台塑已构筑先发壁垒

[R4] 全球钢铁产能过剩，新进入者面临价格战风险

关键解读

1. 产能过剩已成现实
越南总产能~2,850万吨 vs 需求~2,200万吨，过剩约650万吨。VinMetal入场后过剩将超1,150万吨。

2. 和发作为价格锚点
以32%份额成为市场定价者，新进入者必须在其成本线以下才能生存。

3. 出口是唯一出路
三巨头合计远期产能2,640万吨，超出需求~440万吨，倒逼东南亚出口竞争。

4. 高端产品差异化
台塑主攻汽车/家电用钢，和发向高端延伸，VinMetal瞄准VinFast供应链。

和发主导期（2024–2027）

Duality · 二元分析

本土双雄（和发+Vingroup）vs 外来巨鳄（台塑）

两种截然不同的竞争逻辑，在越南3000万吨市场中激烈碰撞

本土阵营 · 和发 + Vingroup

+ 本土政策红利与政府关系深嵌

+ 深耕20年+的经销商网络与客户关系

+ 和发澳洲铁矿保障原料安全

+ Vingroup内需体系（VinFast+Vinhomes）

- 和发高端产品技术积累不足

- Vingroup钢铁经验为零

外来阵营 · 台塑河静

+ 台塑集团60年工业制造基因

+ 全球供应链议价能力（铁矿石/焦煤）

+ 高端汽车板/家电板技术成熟

+ 越欧FTA零关税出口欧洲通道

- 2016年环保事件品牌形象受损

- 河静远离南方消费市场的物流成本

Strategic Insight
台塑的技术优势在长周期中将被和发追赶，和发的本土优势在台塑深耕20年后也在消解。真正的变量是Vingroup的入场速度与规模。

Mega Trends · 结构性趋势

重塑越南钢铁业的四大结构性趋势

需求增长、绿色转型、贸易保护、供应链区域化 — 四个不可逆的力量

1
越南钢铁需求持续增长
2024年人均钢消费~240kg，远低于中国650kg、韩国1,100kg。城市化率42%→目标50%+，基建投资GDP占比~6%。预计2030年需求达3,000-3,500万吨。
2
绿色钢铁转型压力
欧盟CBAM碳边境税2026年全面实施，出口欧洲钢材需为碳付费。和发/台塑高炉工艺碳排放高，氢基直接还原铁(DRI)技术投资门槛高。绿色转型成本$10-20亿级别。
3
全球贸易保护主义升温
美国232条款、欧盟保障措施、东南亚多国反倾销调查频发。越南钢铁出口面临日益严峻的贸易壁垒。2024年多国对越南钢铁发起反倾销调查增加40%。
4
供应链区域化重构
"China+1"战略加速，越南成为制造业转移核心目的地。汽车、家电、机械等下游产业聚集，拉动高端钢材需求。区域供应链整合创造新的竞争维度。

Strategy · 战略建议

三家公司差异化战略路径

基于竞争位势与趋势研判，为三家公司提出差异化战略建议

和发集团
守成者 → 升级者

1. 产品升级：加速汽车板、电工钢等高端产品研发，缩小与台塑技术差距

2. 绿色转型：率先布局DRI/电弧炉路线，抢占绿色钢铁先机应对CBAM

3. 出口扩张：利用CPTPP/越欧FTA扩大东盟与欧洲出口，消化过剩产能

4. 防守策略：在VinMetal投产前锁定核心客户长协，构筑转换成本壁垒

台塑河静
挑战者 → 差异化者

1. 高端聚焦：锁定汽车板、家电板高端市场，不与和发正面价格竞争

2. 二期审慎：仅在确认出口市场（欧洲/东南亚）充足需求后启动二期

3. ESG修复：持续投入环保与社区关系修复，打造绿色钢铁品牌形象

4. 物流破局：投资南部仓储/加工中心，弥补河静地理劣势

Vingroup VinMetal
新进入者 → 颠覆者

1. 错位竞争：起步期避开和发长材/台塑板材，聚焦VinFast供应链需求

2. 技术合作：引入日韩/欧洲钢铁巨头技术合资，弥补经验短板

3. 节奏控制：一期300-500万吨分阶段投产，避免一次性大规模冲击市场

4. 现金流管理：VinFast亏损背景下钢铁业务独立融资，降低集团风险

Strategic Priority
三家公司的最优策略路径截然不同：和发应"进攻性升级"，台塑应"防守性差异化"，Vingroup应"颠覆性错位"。

Conclusion · 核心结论

越南钢铁业正从双雄对峙走向三国杀，
但真正的分水岭不在产能规模，
而在绿色转型与高端化的速度。

和发守成但不升级则被超越，台塑不差异化则陷入价格战泥潭，Vingroup不控制节奏则拖累整个集团。三家公司的最优路径各不相同，但共同面对一个终极命题：在产能严重过剩的市场中，如何找到不可替代的存在价值。
